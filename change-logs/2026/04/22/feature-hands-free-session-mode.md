Shipped the hands-free session mode — the new core UX contract. One-click "Start session" enters an automatic OBSERVING → ARMED → RUNNING → FINISHED → 15-second COOLDOWN → OBSERVING loop that the rider never has to interrupt between runs. Auto re-arm waits for the ROI to be empty and still for 2 s before rebuilding the reference frame, which compensates for ambient-light drift and prevents tainted-reference false starts. Daylight-readable signal colours (lime "go", amber "wait", coral "error/come over") drive the whole body background so the rider can read the phone's state from 10 m under bright sun; ARMED gently pulses. The big route timer keeps the last run's value displayed through cooldown/observing/armed and only resets when the next run actually starts; a small sub-line renders the 15 s countdown, "Ready to go", or previous-run time depending on phase. Retired the old Arm / Cancel / New run button cycle in favour of a single "Start session" / "Stop session" toggle; controls hide during the session and re-appear briefly on tap.
