Added GitHub Actions workflow at `.github/workflows/pages.yml` that publishes the repo root to GitHub Pages on every push to `master`. Enables a stable HTTPS origin so the PWA can be installed once on a phone and run fully offline without keeping the local dev server alive. No source changes were needed — all asset references in `index.html`, `manifest.json`, and the service worker are already relative, so the app works at any sub-path (e.g. `https://<owner>.github.io/Gymkhana-Timer/`). The local `bun run start` dev server remains unchanged for on-the-fly development.
