Set up dev-3.0 project config and add a cloudflared quick tunnel + in-terminal QR code to the dev server.

`.dev3/config.json` now wires `DEV3_PORT0` into the project's `PORT` env var (`scripts/dev-server.ts` reads it) and declares `setupScript: bun install`, `clonePaths: [node_modules]`, `defaultBaseBranch: master`, `portCount: 1`, plus preserves the `builtinColumnAgents.review-by-ai` block that the dev-3.0 UI had already written to master. `.gitignore` also gains `.dev3/config.local.json`. Every `bun run dev` now spawns `cloudflared tunnel --url http://localhost:$PORT`, prints the assigned `trycloudflare.com` HTTPS URL, and renders it as a scannable QR code via the `qrcode-terminal` dev dependency — camera + service worker need HTTPS, so this replaces the manual "open a second terminal, run ngrok, copy URL" dance. Opt-out via `TUNNEL=0`. See `decisions/002-dev-server-cloudflared-tunnel.md`.
